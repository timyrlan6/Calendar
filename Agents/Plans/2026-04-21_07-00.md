## Принятые решения

### 1. GUID для публичной ссылки
- **Требование:** GUID генерируется клиентскими средствами C# через `Guid.CreateVersion7()`
- **Конфигурация EF:** `.ValueGeneratedNever()` — чтобы EF не перезаписывал значение из C#
- **Почему Version 7:** sequential GUID, оптимизирован для PostgreSQL

### 2. Timezone
- Брать из браузера пользователя: `Intl.DateTimeFormat().resolvedOptions().timeZone`
- Хранить как IANA timezone string в ApplicationUser.TimeZoneId

### 3. Публичная ссылка
- URL: `/booking/{guid}`
- Несуществующий GUID → 404

### 4. TimeOnly поля в БД
- Тип: `time without time zone` в PostgreSQL

### 5. Register.razor
- Создать ApplicationUser + BookingLinkId (Guid.CreateVersion7())
- Создать EventType (30 min, default)
- В одной транзакции (SaveChanges)

## ✅ Принятые решения (утверждено 2026-04-21)

> **ВАЖНО:** Эти решения зафиксированы и не подлежат пересмотру.

| Решение | Значение |
|---------|---------|
| EventType | Поддержка разных типов событий с разной длительностью |
| GUID | `Guid.CreateVersion7()` на клиенте + `.ValueGeneratedNever()` в EF |
| EXCLUDE constraint | `BookingLinkId` + `tstzrange(StartTimeUtc, EndTimeUtc) WITH &&` |
| Timezone | IANA string из браузера (`Intl.DateTimeFormat().resolvedOptions().timeZone`) |
| Публичная ссылка | URL: `/booking/{guid}`, несуществующий GUID → 404 |
| Рабочие часы | 14 полей TimeOnly (Mon-Sun Start/End) |
| Register | Создаёт BookingLinkId + EventType (30 min, default) в одной транзакции |

### Алгоритм генерации слотов (c EventType)

```csharp
var eventTypes = await context.EventTypes
    .Where(e => e.BookingLinkId == ownerBookingLinkId)
    .ToListAsync();

var dayOfWeek = date.DayOfWeek;
var (start, end) = dayOfWeek switch
{
    DayOfWeek.Monday => (user.MonStart, user.MonEnd),
    DayOfWeek.Tuesday => (user.TueStart, user.TueEnd),
    DayOfWeek.Wednesday => (user.WedStart, user.WedEnd),
    DayOfWeek.Thursday => (user.ThuStart, user.ThuEnd),
    DayOfWeek.Friday => (user.FriStart, user.FriEnd),
    DayOfWeek.Saturday => (user.SatStart, user.SatEnd),
    DayOfWeek.Sunday => (user.SunStart, user.SunEnd),
    _ => (TimeOnly.MinValue, TimeOnly.MinValue)
};

if (start == TimeOnly.MinValue || end == TimeOnly.MinValue)
    return [];

var tz = TimeZoneInfo.FindSystemTimeZoneById(user.TimeZoneId);
var localStart = date.Date + start.ToTimeSpan();
var utcStart = TimeZoneInfo.ConvertTimeToUtc(localStart, tz);

var slots = new List<DateTime>();
foreach (var eventType in eventTypes)
{
    var duration = eventType.Duration;
    for (var t = utcStart; t < utcEnd; t = t.AddMinutes(duration.TotalMinutes))
    {
        slots.Add(t);
    }
}

return slots;
```
