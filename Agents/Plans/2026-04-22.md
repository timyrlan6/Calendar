# Запись на звонок — Итог всех рассуждений

**Дата:** 2026-04-22

## Цель проекта

Упрощенный сервис бронирования времени по мотиву Cal.com на Blazor с PostgreSQL.

---

## ✅ Принятые решения (утверждено, НЕ подлежат пересмотру)

| Решение | Значение |
|---------|----------|
| EventType | Поддержка разных типов событий с разной длительностью |
| GUID | `Guid.CreateVersion7()` на клиенте + `.ValueGeneratedNever()` в EF |
| EXCLUDE constraint | `BookingLinkId` + `tstzrange(StartTimeUtc, EndTimeUtc) WITH &&` |
| Timezone | IANA string из браузера (`Intl.DateTimeFormat().resolvedOptions().timeZone`) |
| Публичная ссылка | URL: `/booking/{guid}`, несуществующий GUID → 404 |
| Рабочие часы | 14 полей TimeOnly (Mon-Sun Start/End) |
| Register | Создаёт BookingLinkId + EventType (30 min, default) в одной транзакции |
| DateTime | Всегда UTC (`DateTimeKind.Utc`), НЕ `Unspecified` |
| Строка подключения | Из appsettings.json проекта (НЕ из внешних инструментов) |
| Текстовые поля | 256 — маленькие, 1024 — большие |

## Fluent API (ApplicationDbContext)

```csharp
builder.Entity<ApplicationUser>(entity =>
{
    entity.HasIndex(e => e.BookingLinkId).IsUnique();
    entity.Property(e => e.BookingLinkId).ValueGeneratedNever();
});

builder.Entity<EventType>(entity =>
{
    entity.HasKey(e => e.Id);
    entity.Property(e => e.Id).ValueGeneratedOnAdd();
    entity.HasIndex(e => e.OwnerId);
    entity.HasIndex(e => e.BookingLinkId);
    entity.HasOne(e => e.Owner)
        .WithMany(u => u.EventTypes)
        .HasForeignKey(e => e.OwnerId)
        .OnDelete(DeleteBehavior.Cascade);
});

builder.Entity<Booking>(entity =>
{
    entity.HasKey(e => e.Id);
    entity.HasIndex(e => e.BookingLinkId);
    entity.HasOne(e => e.Owner)
        .WithMany(u => u.Bookings)
        .HasForeignKey(e => e.OwnerId)
        .OnDelete(DeleteBehavior.Cascade);
});
```

## EXCLUDE Constraint

Для защиты от пересекающихся бронирований:

```sql
CREATE EXTENSION IF NOT EXISTS btree_gist;
ALTER TABLE "Bookings" ADD CONSTRAINT "no_overlap" EXCLUDE USING gist (
    "BookingLinkId" WITH =,
    tstzrange("StartTimeUtc", "EndTimeUtc") WITH &&
);
```

**Важно:** Нужен extension `btree_gist` для GIST с uuid.
